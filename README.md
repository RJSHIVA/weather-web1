# React + Vite Template

This template provides a minimal setup to get React working in Vite with Hot Module Replacement (HMR) and some ESLint rules.

## Official Plugins

This project utilizes two official plugins:

1. **[@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md)**
   - Uses [Babel](https://babeljs.io/) for Fast Refresh.
   - [Plugin Documentation](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md)

2. **[@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc)**
   - Uses [SWC](https://swc.rs/) for Fast Refresh.
   - [Plugin Documentation](https://github.com/vitejs/vite-plugin-react-swc)

## Getting Started

1. **Clone this repository:**

    ```bash
    git clone https://github.com/rjshiva/weather-web1.git
    cd weather-web1
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Run the development server:**

    ```bash
    npm run dev
    ```

   This will start the development server at `http://localhost:3000`. Open your browser and navigate to this URL to see your React app in action.

## Available Scripts

- **npm run dev:** Run the development server with HMR.
- **npm run build:** Build the project for production.
- **npm run serve:** Serve the production build for testing.
